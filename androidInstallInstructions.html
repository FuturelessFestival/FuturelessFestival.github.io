Download APK



When you press the download link above or at the bottom of this page, you'll likely recieve the following message:

Press "Download anyway" to download our APK file which is a "Android package file" that contains our app.

When the file is done downloading, press "Open".

If the phone tells you that your phone is not allowed to install unknown apps from this source press "SETTINGS".



You will now be able to allow installing from that source, turn this on by pressing on the switch.



Press the arrow to go backwards.



Press install, and when it's done press open
